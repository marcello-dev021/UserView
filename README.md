# 👥 UserView - API de Clientes

Aplicação web que consome uma API pública de clientes e exibe os dados em uma interface gráfica interativa.  
O projeto demonstra integração com APIs externas, manipulação de dados assíncronos e renderização dinâmica utilizando JavaScript, HTML e CSS.

---

## 🚀 Funcionalidades

- 🔄 Consumo de API pública
- 📋 Listagem dinâmica de clientes
- 🔎 Filtro de clientes em tempo real
- 🎨 Interface organizada com HTML e CSS
- ⚡ Programação assíncrona com async/await

---

## 🛠️ Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript (ES Modules)
- Fetch API
